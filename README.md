# Tonic Events

## Usage

`https://raw.githubusercontent.com/tonic-cash/events/main/${chainId}/${contractAddress}.json`

- `fromBlock`: `number`
- `endBlock`: `number`
- `events`
  - `returnValues`
    - `0`
    - `1`
    - `2`
    - `commitment`
    - `leafIndex`
    - `timestamp`
  - `blockNumber`: `number`
